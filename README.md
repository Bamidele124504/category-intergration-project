# Category Integration Project

## 📌 Overview
This is a full-stack application built with:

- NestJS (Backend)
- Angular (Frontend)
- PostgreSQL (Database)

---

## 🚀 Features
- User authentication (JWT)
- Create products with category selection
- One-to-many relationship (Category → Products)
- Dynamic category dropdown
- Protected API endpoints

---

## 📁 Project Structure

- `products-api` → Backend (NestJS)
- `product-cart-app` → Frontend (Angular)

---

## ⚙️ Setup
### Frontend
```bash
cd product-cart-app
npm install
ng serve 
 ```
### Backend
```bash
cd products-api
npm install
npm run start:dev
 ```
### API
/api/v1/auth/login
/api/v1/categories
/api/v1/products
